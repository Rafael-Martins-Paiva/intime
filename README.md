# intime

Este projeto é um simples chat room, usando express.

## comandos para baixar o projeto

você vai precisar rodar os seguintes comandos

``` bash
npm install
```


## comandos para rodar o projeto

você vai precisar rodar os seguintes comandos

``` bash
npm run dev
```

Verifique se você possui:

Node.js instalado (versão 14 ou superior)
O npx serve funciona normalmente em versões recentes do Node.js, então recomenda-se Node.js ≥ 14 para melhor compatibilidade.

NPM (já vem junto com o Node.js)
